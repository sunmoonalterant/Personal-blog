民生数治惠家是一套面向社区居民、网格员与管理者的数字化治理 Demo。项目以微信小程序作为主要服务入口，连接居民参与、网格响应、后台管理与风险预警，帮助社区将日常服务与基层治理任务沉淀为可追踪的协同闭环。

## 核心能力

- **居民服务**：提供公告通知、社区活动报名与签到、积分累计、商城兑换和个人中心等功能。
- **网格员工作台**：支持走访记录、工单处理、预警查看、任务看板和高风险老人管理。
- **管理后台**：基于 Vue 构建用户、活动、积分规则、工单、预警与运营数据看板。
- **主动预警**：Python 规则引擎识别连续 3 天未签到的黄色预警，以及连续 7 天未签到的红色预警，辅助网格员及时跟进。

## 技术架构

系统采用多端协同架构：微信小程序服务居民端与网格员端，Vue 负责管理后台，Spring Boot 对外提供 REST API，并连接 MySQL、Redis 与 Python 预警服务。当前前端默认提供 Mock 数据，便于在离线环境中演示完整的社区治理流程。

## 项目价值

通过“居民参与 → 网格响应 → 后台管理 → 风险预警”的工作闭环，项目将分散的社区服务信息整合为可视化、可追踪的治理任务，提升社区服务效率与基层治理的主动性。
